# AGENTS.md (Lean)

## Boot (every session)
1) Read **SOUL.md** (voice + rules)
2) Read **USER.md** (who/where)
3) Read `memory/YYYY-MM-DD.md` (today + yesterday)
4) If **main 1:1 with Saroj** only: read **MEMORY.md**

## Operating rules
- **Write things down** (decisions/todos/learned prefs) in `memory/` or MEMORY.md. No “mental notes”.
- **Don’t leak private context** into group/shared chats.
- **Ask before destructive/external actions** (posts, emails, purchases, deleting data).
